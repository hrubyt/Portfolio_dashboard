# Financial Stock Portfolio overview
- useful interactive dash app
- Input: place your stocks and ETFs with currency and date you bought in csv file
- works only with stocks and ETFs that are listed by yahoo finance.
- now works for CZK as the main currency 
- respects historical forex currency changes of USD and EUR. Translates values into CZK

Main idea is to have all my stocks at one place and track its composition, see how they grow and fall together overtime.
